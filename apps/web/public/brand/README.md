# TaxiLink Pro — Pack d'identité visuelle

## Charte couleurs

```css
--tl-black:   #09090B;
--tl-yellow:  #FBBF24;
--tl-white:   #FFFFFF;
--tl-gray:    #71717A;
--tl-gray-d:  #A1A1AA;
```

Typographie : **Plus Jakarta Sans** (weight 800 pour TaxiLink, 700 pour Pro).

## Règles d'usage

- Espace de respiration : laisser au minimum la hauteur du badge « Pro » autour du logo
- Taille minimum : 24px pour l'icône seule, 120px de large pour le lockup horizontal
- Ne pas : changer les couleurs, déformer les proportions, ajouter ombres/dégradés/contours
- Sur photo : utiliser `logo-dark.svg` ou poser un rectangle blanc de protection
- Tagline : ne jamais séparer « Partagez vos courses. » et « Gagnez plus. »
